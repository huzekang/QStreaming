 ## 编译项目
 移动到项目根目录。
 执行：
 ```
 mvn clean install -DskipTests  
```

## 本机测试执行SQL作业

## 启动hive support
```
spark-submit \
--class com.qiniu.stream.core.Streaming \
--master local[*]  \
stream-standalone/target/stream-standalone-0.2.0-SNAPSHOT-jar-with-dependencies.jar \
-j examples/hive2console.dsl -c stream.hive.enable=true
```

### 批处理-写入hudi数据源
```shell script
spark-submit \
--class com.qiniu.stream.core.Streaming \
--master local[2] \
--conf 'spark.serializer=org.apache.spark.serializer.KryoSerializer' \
--jars stream-hudi/target/stream-hudi-0.2.0-SNAPSHOT-jar-with-dependencies.jar \
stream-standalone/target/stream-standalone-0.2.0-SNAPSHOT-jar-with-dependencies.jar \
-j examples/hdfs2hudi.dsl
```
